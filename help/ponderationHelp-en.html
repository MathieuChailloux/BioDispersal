Ponderation


This step is optional.

Ponderation is a way to refine friction layers.

Ponderation mode

  • Direct : friction and ponderation layers are multiplied
  • Intervals : friction pixel is multiplied by ponderation value if ponderation pixel belongs to matching interval, and remains unchanged if no interval matches.
  • Buffers : friction pixel is multiplied by ponderation value if it belongs to a buffer area around ponderation layer specified by upper and lower bounds (distance in meters).
  • Maximum : result is the maximal value of friction and ponderation pixels.
  • Minimum : result is the minimal value of friction and ponderation pixels.
