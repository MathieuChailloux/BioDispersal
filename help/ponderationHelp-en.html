Weighting


This step is optional.

Weighting is a way to refine friction layers.

Weighting mode

  • Direct : friction and ponderation layers are multiplied
  • Intervals : friction layer pixel is multiplied by ponderation value if ponderation layer pixel belongs to matching interval, and remains unchanged if no interval matches.
  • Buffers : friction layer pixel is multiplied by ponderation value if it belongs to a buffer area around ponderation layer specified by upper and lower bounds (distance in meters).
  • Maximum : result is the maximal value of friction and ponderation layers pixels.
  • Minimum : result is the minimal value of friction and ponderation layers pixels.
