Sous-trames


Cette étape consiste à déclarer les sous-trames qui seront considérées dans le projet
Une sous-trame est un type d'habitat qui sera l'objet d'une modélisation spécifique.
Le nom de la sous-trame doit être un identifiant en un seul mot, la description permet de détailler son contenu.
Pour chaque sous-trame, un dossier spécifique sera constitué par les étapes suivantes avec couches de fusion, de friction, de pondération et de dispersion.
