Selection


The purpose of selection step is to extract and standardize input data. A selection action is performed from an input layer according to a selection mode. A group and a class are assigned to each selection result.


Input layer
Input layer format (raster or vector) mus be set.
Input layer must be loaded in QGIS project to be available in BioDispersal.
Input layer must be in Shapefile (.shp) or GeoTIFF (.tif) format.

Selection mode

  • Vector
    • By field: input layer entities are classified accorindg to selected field values.
    • By expression: input layer entities verifying the expression are selected. If expression is empty, every entity is selected.
  • Raster : resampling mode matches gdalwarp option -r.
    • Near: to select for discrete data
    • Mean : to select for continuous data

Groups
Merge groups gather data with same geometry (point, line, polygon or raster) according to thematic criterias (land use, roads, ...).

Classes
Friction classes are automatically generated by BioDispersal.
