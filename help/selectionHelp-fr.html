Sélection


La sélection consiste à extraire et uniformiser les données d'entrée. Une sélection s'effectue depuis une couche d'entrée selon un mode et affecte un groupe et une classe au résultat.


Couche d'entrée
Le format (raster ou vecteur) de la couche d'entrée doit être renseigné.
La couche doit être chargée dans le projet QGIS pour être sélectionnée.
Seuls les formats Shapefile (.shp) et GeoTIFF (.tif) sont acceptés pour l'instant.

Mode de sélection

  • Vecteur
    • Par champ : les entités de la couches sélectionnées sont regroupées selon les valeurs du champ sélectionné.
    • Par expression : sélectionne les entités selon l'expression renseignée. Si l'expression est vide, toutes les entités sont sélectionnées.
  • Raster : le mode de ré-échantillonage correspond à l'option -r de gdalwarp.
    • Plus proche voisin : à sélectionner pour une donnée discrète.
    • Moyenne : à sélectionner pour une donnée continue.
